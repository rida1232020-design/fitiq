# FitIQ العراق 🏋️🇮🇶

## 📱 مميزات التطبيق

### للرياضي (Athlete)
- ✅ تسجيل حساب جديد
- 🏆 استعراض قائمة الكباتن المتاحين
- 💬 تواصل مباشر مع الكابتن (شات حقيقي)
- 📚 استعراض والاشتراك بالكورسات
- 💪 مكتبة تمارين كاملة مع نصائح
- 🏋️ جلسات تمرين تفاعلية مع مؤقت

### للكابتن (Captain)
- ✅ تسجيل وإنشاء ملف احترافي
- 📚 إضافة وتعديل وحذف الكورسات
- 💬 استقبال رسائل الرياضيين والرد عليهم
- 🟢 تغيير حالة الإتاحة (متاح/غير متاح)
- 📊 إحصائيات: عدد الكورسات، المحادثات، التقييم

## 🏗️ هيكل المشروع

```
src/
├── lib/
│   ├── supabase.js      # اتصال قاعدة البيانات
│   └── theme.js         # ألوان وستايلات
├── context/
│   └── AuthContext.js   # إدارة المستخدم
├── components/
│   ├── UI.js            # مكونات مشتركة
│   └── ChatRoom.js      # شات حقيقي realtime
└── pages/
    ├── AuthPage.js         # صفحة الدخول والتسجيل
    ├── AthleteApp.js       # تطبيق الرياضي
    └── CaptainDashboard.js # لوحة تحكم الكابتن
```
